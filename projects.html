Doge Daycare for Android 4.0.3 and above

Android app that I'm developing for use in a Dog Daycare / Dog Boarding business.

Some features

PIN code system for login

Access restrictions to employee database. Only administrator accounts can view and modify employee information.
Accounts can be elevated or stripped of administrator access.

Animal vaccination date tracking

Check if an animal has gotten their :
3 year vaccinations
         Rabies
         Distemper (VDC)
         Adenovirus / Canine Hepatitis (CAV-2)
         Parainfluenza (CPiV)
         Parvovirus (CPV2)
6 to 12 month vaccinations
         Bordetella (Bb)

Information database of employees, animals, and appointments.

Easy to backup SQLite database.
Keep track of :
         Unique IDs for each entry
         First and Last name
         Email address
         Home address
         Phone number
         Start & End dates of appointments
